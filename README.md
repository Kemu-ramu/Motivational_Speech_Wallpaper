# User Card App

名言やテキストを背景画像付きのカードとして表示するアプリケーション

## 機能

- テキスト、文字色、背景画像のカスタマイズ
- テキストの配置位置（上下左右中央）の指定
- 複数のカードを一括生成・表示

## 使い方

`index.html`をブラウザで開いて使用してください。
`WallPaper.js` 内の `wallPaperList` に新しい `WallPaper` インスタンスを追加することで、カードを増やすことができます。

## ファイル構成

- `index.html` - HTMLファイル
- `WallPaper.css` - スタイルシート
- `WallPaper.js` - JavaScript機能
